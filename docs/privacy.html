브랜드브릿지 개인정보처리방침

브랜드브릿지(이하 "앱")는 사용자의 개인정보를 직접 수집하지 않습니다.

1. 수집하는 정보

본 앱은 회원가입 또는 로그인 기능을 제공하지 않습니다. 다만, 앱 내 제휴 링크(예: LinkPrice)를 클릭할 경우 제3자 웹사이트로 이동하며, 해당 제3자가 IP 주소, 기기 정보, 쿠키, 클릭 정보 등을 수집할 수 있습니다.

2. 정보의 이용 목적

제3자가 수집한 정보는 광고 추적, 통계 분석 및 제휴 수익 정산 목적으로 이용될 수 있습니다.

3. 제3자 제공

본 앱은 개인정보를 직접 수집하거나 저장하지 않으며, 제3자 플랫폼의 개인정보 처리에 대해서는 해당 플랫폼의 정책이 적용됩니다.

4. 데이터 보안

본 앱은 HTTPS 보안 연결을 사용합니다.
